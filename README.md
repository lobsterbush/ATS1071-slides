# ATS1071 Tutorial Slides

**Political Theory** · Monash University · Semester 1, 2026

## Authors
Charles Crabtree, Senior Lecturer, School of Social Sciences, Monash University and K-Club Professor, University College, Korea University.

## Overview
Tutorial slides for ATS1071, an introductory political theory course organised around contemporary political problems. Built with [Reveal.js](https://revealjs.com/).

## Contents
- **Week 1** — Introduction (gallery walk + argument mapping)
- **Week 2** — Free Speech (jigsaw + structured controversy)
  - Generic version + class-specific versions (10am, 12pm, 4pm) with embedded survey data

## Usage
Visit the [live site](https://lobsterbush.github.io/ATS1071-slides/) or open any `.html` file directly in a browser. Use arrow keys or swipe to navigate slides.

## Requirements
A modern web browser. All dependencies (Reveal.js, fonts) are loaded via CDN.
